To create a custom policy, you need to be assigned to the Organization Administrators Role.

. xref:create-policy-config-task.adoc[Create a custom policy configuration XML file].
. xref:create-policy-definition-task.adoc[Create a custom policy definition YAML file].
. xref:add-custom-policy-task.adoc[Add the policy to API Manager].
. xref:tutorial-manage-an-api.adoc[Test the policy by applying it to a deployed API].

*Note:* In API Manager 2.x, creating a custom policy has been simplified using Maven. See
xref:2.x@custom-policy-getting-started.adoc[Custom Policy Development Reference].
